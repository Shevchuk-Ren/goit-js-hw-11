# goit-js-hw-11
Создан репозиторий goit-js-hw-09.
Проект собран с помощью parcel-project-template.
Все три задания оформлены в одном стиле.
Создан переключатель цветов - скрипт, который после нажатия кнопки Start, раз в секунду меняет цвет фона <body> на случайное значение используя инлайн стиль. 
При нажатии на кнопку Stop, изменение цвета фона останавливается.
Создан таймер обратного отсчета - скрипт таймера, который ведёт обратный отсчет до определенной даты. 
Если пользователь выбирает дату в прошлом, в модальном окне показывает уведомление "Please choose a date in the future". 
Используй библиотеку sweetalert2 для всплывающего окна.
Кнопка не активна до тех пор, пока пользователь не выбрал дату в будущем.
Если выбрана валидная дата и пользователь нажал кнопку - начинается отсчет времени.
Количество дней может состоять из более чем двух цифр.
Таймер останавливается когда дошел до конечной даты, то есть 00:00:00:00. При этом кнопка становится не активна пока не выберешь новую дату в будущем.
При изменении даты во время начала обратного отсчета, дата меняется на новую, отсчет начинается с учетом выбраной новой даты.
При клике на кнопку во время начала обратного отчеста без изменения даты на новую, отсчет не прерывается.
Промисификация функций - содержит в себе три подзадания, которые нужно сделать с использованием промис.
